# 🍳 Cheffy - AI Recipe Generator

A React-based web application that generates AI-powered recipe suggestions based on ingredients you have at home.

## 🚀 Features
- Add and manage ingredients dynamically
- Generate recipes using AI (Hugging Face API)
- View recipes in clean Markdown format
- Responsive and modern UI

## 🛠️ Tech Stack
- React.js, JavaScript, CSS3
- Vite
- Hugging Face Inference API
- React-Markdown

## ▶️ Run Locally
```bash
npm install
npm run dev
